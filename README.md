## ⚙️ How the App Works

Diabetes-Prediction uses models built with machine learning. These models study patterns from health data collected over time to predict risk. The main methods include:

- Decision Trees  
- Random Forests  
- Logistic Regression  
- Support Vector Machines  
- Gradient Boosting  
- XGBoost  

All these help the app decide how likely you may develop lung cancer based on your personal health data.

The app’s interface is built using Streamlit. This means it runs in a window that looks like a simple website but operates locally on your computer. It does not require you to open a browser or install extra software.

## 📚 Useful Definitions

- **Machine Learning**: A method where computers learn patterns from data to make predictions.  
- **Streamlit**: A tool that creates simple user interfaces for Python apps.  
- **Decision Tree**: A step-by-step model that splits data into groups to make decisions.  
- **Random Forest**: Many decision trees combined to improve accuracy.  
- **Gradient Boosting and XGBoost**: Advanced ways to combine multiple models for better results.  
- **Logistic Regression, Support Vector Classifier**: Types of algorithms used to classify data.  

These terms help describe how the app calculates your risk, but you do not need to know them to use the app.
